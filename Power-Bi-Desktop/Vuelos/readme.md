
# Dashboard de Análisis de Vuelos ✈️

Este proyecto analiza retrasos de vuelos usando **Power BI** y **Machine Learning** (RandomForestRegressor).  
Incluye:
- Dashboard interactivo (`dashboard.pbix`)
- Dataset original (`flights.csv`)
- Modelo entrenado (`champion.pkl`)
- Documentación en PDF (`optimización_aeropuestaria.ipynb - Colab.pdf`)

---

## 📂 Archivos del repositorio

| Archivo | Descripción |
|---------|-------------|
| `dashboard.pbix` | Dashboard en Power BI |
| `flights (1).csv` | Dataset de vuelos |
| `champion.pkl` | Modelo entrenado con scikit-learn |
| `optimización_aeropuestaria.ipynb - Colab.pdf` | Documentación del análisis y modelado |
| `MinimalCeleste.json` | Tema personalizado para Power BI |
